# Neko Rogaru music prompts

These prompts are written for a general AI music generator. Generate instrumental music only and request permission for commercial use from the provider you choose.

## Main gameplay loop

> Create a seamless instrumental gameplay loop for a cute vertical pixel-art web game called Neko Rogaru, where a round tabby cat tumbles endlessly down forests, rivers, shrine paths, and magical caves. Playful, nimble, warm, and slightly chaotic—but never stressful. 144 BPM, G-major pentatonic, 48 bars, approximately 80 seconds. Blend polished 16-bit game music with koto-like plucks, marimba, soft woodblocks, tiny shakers, round synth bass, gentle chiptune arpeggios, and a very occasional airy flute phrase. Use one memorable four-note cat-like hook and light syncopation to create falling momentum. Keep the middle frequencies uncluttered so pickup and collision sound effects remain clear. No vocals, no lyrics, no copyrighted melody, no cinematic trailer drums, no harsh lead synth, no long intro, no fade-out. The final bar must connect perfectly back to the first bar.

## Matched time-of-day variations

Ask for all four versions in the same key, tempo, bar count, arrangement structure, and exact loop points so the game can crossfade between them.

### Morning

> Re-arrange the Neko Rogaru main loop as a fresh woodland morning. Warm marimba, soft koto plucks, brushed shaker, tiny birdlike flute answers, light bass, lots of air. Curious and optimistic, with restrained percussion. Preserve the melody, 144 BPM, G-major pentatonic, 48 bars, and seamless loop.

### Noon

> Re-arrange the same Neko Rogaru loop for bright noon. Stronger woodblock groove, bouncier bass, sparkling chiptune arpeggios, and the clearest version of the four-note hook. Energetic and playful without becoming frantic. Preserve the exact tempo, key, length, structure, and loop points.

### Sunset

> Re-arrange the same loop for a cozy orange sunset. Softer percussion, mellow plucked strings, warm rounded synth pads, and a slightly nostalgic counter-melody. Still moving forward, never sleepy or sad. Preserve the exact tempo, key, length, structure, and loop points.

### Night

> Re-arrange the same loop for a magical moonlit forest. Muted plucks, glassy bells used sparingly, soft low marimba, subtle firefly-like arpeggios, and gentle mysterious harmony. Cute wonder rather than darkness or danger. Preserve the exact tempo, key, length, structure, and loop points.

## Interaction cues

Generate these as separate short files using the same sound palette and key.

- **Branch rest:** “An 8-bar suspended rest cue for a cat hanging safely from a branch. Half the percussion drops away, the four-note hook pauses on a playful unresolved note, and the cue can loop indefinitely before snapping cleanly back into the main track.”
- **Catnip boost:** “A 4-bar speed-up layer with quick chiptune arpeggios, shaker rush, and rising koto glissando. Exciting and cute, not aggressive.”
- **Discovery:** “A bright one-second musical sparkle ending with a tiny two-note meow-shaped phrase.”
- **Bonk:** “A funny half-second wooden percussion bonk with a soft descending pluck. Comic, harmless, and not painful.”
- **Cute ending:** “A two-second surprise landing sting: quick tumble flourish, tiny pause, then a warm resolved chord and playful bell.”

## Export request

Request 48 kHz WAV masters plus compressed OGG or MP3 versions. If supported, also request separate percussion, melody, bass, and ambience stems. Do not let the generator add silence or a fade at either end of a seamless loop.
